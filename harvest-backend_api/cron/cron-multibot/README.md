# Cron Multi-bot

Can be used to periodically run small tasks.
Currently used for:

1. Daily profit notification
2. Claiming sETH2 rewards

## Setup

1. set up `settings.json` and `secret.json`
1. `bash run-standalone.sh`

## `settings.json`
```
{
  "gasPriceMax": "201000000000",
  "gasLimit": "3000000"
}
```
